# Ghostbox API

Use Ghostbox MCP tools instead of shelling out for host features.

Available tools:
- `mcp__ghostbox__mailbox` - check inbox, read mail, send mail, reply
- `mcp__ghostbox__schedule` - create, list, and delete scheduled prompts
- `mcp__ghostbox__memory_write` - append to `MEMORY.md` or `USER.md`
- `mcp__ghostbox__memory_replace` - replace a matching memory entry
- `mcp__ghostbox__memory_remove` - remove a matching memory entry
- `mcp__ghostbox__memory_show` - inspect current memory contents and usage

Use these tools directly when you need mailbox, scheduling, or warm-memory operations.
